Lesson 10: Fungsi Keanggotaan Himpunan Fuzzy

7 jenis kurva untuk menerjemahkan nilai numerik ke derajat keanggotaan 0–1

1. Apa itu Fungsi Keanggotaan?

Definisi

Fungsi keanggotaan (membership function) adalah kurva matematis yang memetakan titik-titik input (nilai x) ke nilai/derajat keanggotaan dalam interval 0 sampai 1.

Lesson 9 sudah membahas konsep dasarnya. Sekarang pertanyaannya: kurva mana yang dipakai? Tergantung:

  • Berapa parameter domain yang tersedia (1, 2, 3, atau 4 titik)
  • Apakah himpunan ada di ujung, tengah, atau mencakup seluruh variabel
  • Seberapa mulus transisi yang diinginkan (linear vs kurva)

Berikut 7 jenis yang akan kita bahas, diurutkan dari yang paling sederhana sampai yang paling eksotis:

2. Linear Naik (Linear Up)

1Karakteristik: Garis lurus dari derajat 0 di titik a, naik linear ke derajat 1 di titik b.
μ[x] = 0;              jika x ≤ a
μ[x] = (x − a) / (b − a);  jika a ≤ x ≤ b
μ[x] = 1;              jika x ≥ b

Gambar 1: Kurva linear naik — derajat keanggotaan bertambah secara proporsional dengan x.

Contoh: Himpunan PANAS pada variabel temperatur (a=25, b=35)

Himpunan PANAS didefinisikan naik dari 25°C (derajat 0) ke 35°C (derajat 1).

Berapa μPANAS[32]?

μPANAS[32] = (32 − 25) / (35 − 25) = 7/10 = 0,7

Artinya: pada 32°C, seseorang "agak panas" dengan derajat 0,7.

3. Linear Turun (Linear Down)

2Karakteristik: Garis lurus dari derajat 1 di titik a, turun linear ke derajat 0 di titik b. Kebalikan dari linear naik.
μ[x] = 1;              jika x ≤ a
μ[x] = (b − x) / (b − a);  jika a ≤ x ≤ b
μ[x] = 0;              jika x ≥ b

Gambar 2: Kurva linear turun — derajat keanggotaan berkurang secara proporsional dengan x.

Contoh: Himpunan DINGIN pada variabel temperatur (a=15, b=30)

Himpunan DINGIN didefinisikan turun dari 15°C (derajat 1) ke 30°C (derajat 0).

Berapa μDINGIN[20]?

μDINGIN[20] = (30 − 20) / (30 − 15) = 10/15 = 0,667

Artinya: pada 20°C, seseorang masih "agak dingin" dengan derajat 0,667.

4. Kurva Segitiga (Triangular)

3Karakteristik: Kombinasi linear naik + linear turun, membentuk segitiga dengan puncak bernilai 1. Ditentukan oleh 3 titik: a (kaki kiri), b (puncak), c (kaki kanan).
μ[x] = 0;                      jika x ≤ a atau x ≥ c
μ[x] = (x − a) / (b − a);      jika a ≤ x ≤ b
μ[x] = (c − x) / (c − b);      jika b ≤ x ≤ c

Gambar 3: Kurva segitiga — satu titik puncak (b) di mana keanggotaan mencapai 1.

Contoh: Himpunan NORMAL pada temperatur (a=15, b=25, c=35)

Berapa μNORMAL[23]? Karena 15 ≤ 23 ≤ 25, kita pakai rumus sisi kiri:

μNORMAL[23] = (23 − 15) / (25 − 15) = 8/10 = 0,8

Artinya: pada 23°C, derajat keanggotaan "normal" adalah 0,8 (mendekati puncak).

5. Kurva Trapesium (Trapezoidal)

4Karakteristik: Modifikasi segitiga yang memiliki area puncak datar (bernilai 1 dari b ke c). Ditentukan oleh 4 titik: a (kaki kiri), b (puncak kiri), c (puncak kanan), d (kaki kanan).
μ[x] = 0;                      jika x ≤ a atau x ≥ d
μ[x] = (x − a) / (b − a);      jika a ≤ x ≤ b
μ[x] = 1;                      jika b ≤ x ≤ c  (puncak datar)
μ[x] = (d − x) / (d − c);      jika c ≤ x ≤ d

Gambar 4: Kurva trapesium — ada "plateau" di puncak dengan keanggotaan penuh 1.

Contoh: NORMAL trapesium (a=15, b=24, c=27, d=35)

Berapa μNORMAL[32]? Karena 27 ≤ 32 ≤ 35, kita pakai sisi kanan:

μNORMAL[32] = (35 − 32) / (35 − 27) = 3/8 = 0,375

Artinya: pada 32°C, masih "agak normal" (0,375), di luar rentang puncak 24–27.

6. Kurva Bentuk Bahu (Shoulder)

5Karakteristik: Untuk himpunan di ujung variabel, di mana nilai tidak berubah lagi setelah batas ekstrem. Ada dua varian: Bahu Kiri (untuk batas bawah) dan Bahu Kanan (untuk batas atas).

6a. Bahu Kiri (Left Shoulder) — Contoh: DINGIN

μ[x] = 1;              jika x ≤ a  (datar penuh di kiri)
μ[x] = (b − x) / (b − a);  jika a ≤ x ≤ b
μ[x] = 0;              jika x ≥ b

6b. Bahu Kanan (Right Shoulder) — Contoh: PANAS

μ[x] = 0;              jika x ≤ a
μ[x] = (x − a) / (b − a);  jika a ≤ x ≤ b
μ[x] = 1;              jika x ≥ b  (datar penuh di kanan)

Gambar 5: Bahu kiri (DINGIN, datar 1 di kiri) dan bahu kanan (PANAS, datar 1 di kanan). Perhatikan: a di Bahu Kiri = ujung kanan flat-1 (turun mulai), a di Bahu Kanan = ujung kiri rise (naik mulai).

Kapan Pakai Bahu?

Gunakan bahu ketika himpunan ada di ekstrem dari variabel dan tidak masuk akal untuk didefinisikan "naik" atau "turun" di luar batas itu. Contoh: pada variabel temperatur [0, 40], himpunan DINGIN dan PANAS idealnya pakai bahu.

7. Koordinat Keanggotaan (Diskret)

6Karakteristik: Representasi diskret — himpunan didefinisikan sebagai daftar pasangan berurutan (skalar, derajat). Tidak ada rumus kontinu.
Himpunan = { skalar1/derajat1,  skalar2/derajat2,  ...,  skalarn/derajatn }
Contoh: PENGENDARA BERISIKO TINGGI (variabel umur)

Sistem asuransi kendaraan motor. Kurva berbentuk 'U' (risiko tinggi di usia muda DAN tua):

{ 16/1.0, 21/0.6, 28/0.3, 68/0.3, 76/0.5, 80/0.7, 96/1.0 }

Artinya:

  • Pengendara 16 tahun: risiko = 1,0 (maksimal, masih sangat muda)
  • Pengendara 28 tahun: risiko = 0,3 (aman, sudah dewasa)
  • Pengendara 96 tahun: risiko = 1,0 (maksimal, sudah sangat tua)
Kapan Pakai Representasi Diskret?

Ketika data historis tidak kontinu (misalnya hasil survey, database pelanggan) atau ketika pakar mendefinisikan risiko secara eksplisit pada titik-titik tertentu. Untuk nilai x yang berada di antara dua titik, biasanya dilakukan interpolasi linear.

8. Representasi Non-Linear (S-Curve & Lonceng)

Untuk pemetaan dengan kelandaian yang lebih mulus dibanding garis lurus, tersedia dua keluarga kurva non-linear:

8a. Kurva S (Sigmoid)

Kurva berbentuk huruf S, transisi halus dari 0 ke 1. Cocok untuk model pertumbuhan/logistik yang dimulai lambat, akselerasi, lalu saturasi.

8b. Kurva Lonceng (Bell-shaped)

Berbentuk lonceng simetris, mencapai puncak 1 di tengah. Tiga varian populer:

  • Kurva Pi — mirip segitiga tapi dengan kurva di ujungnya
  • Kurva Beta — lebih fleksibel, bisa asimetris
  • Kurva Gauss — distribusi normal, simetris sempurna
Fokus UAS

Untuk soal UAS, yang paling sering keluar adalah linear, segitiga, trapesium, dan bahu. S-Curve dan lonceng cukup dipahami konsepnya saja — dipakai di aplikasi yang butuh distribusi halus (misalnya pengenalan pola).

3. Anda ingin himpunan DINGIN pada variabel temperatur [0, 40] yang didefinisikan "datar di 1" untuk x ≤ 15 dan turun ke 0 di x=20. Fungsi apa yang tepat?
