Horn Form (Forward/Backward Chaining)

Definite clause: disjunction dengan tepat satu literal positif.

Horn clause: maksimal satu literal positif.

Horn: ¬A ∨ ¬B ∨ C ≡ A ∧ B ⇒ C
// Semua aturan Wumpus bisa ditulis Horn!

CNF Resolution

Steps to CNF:

  1. Eliminate ⇔ (replace with ⇒ both ways)
  2. Eliminate ⇒ (¬α ∨ β)
  3. Push ¬ inward (De Morgan's)
  4. Distribute ∨ over ∧

Resolution rule: α∨β, ¬β∨γ ⊢ α∨γ

SP Bagian 6: Inferensi & Anomali Logika

Forward Chaining (Data-Driven)

Arah: Fakta → Konklusi (maju, divergen)

Cocok: Banyak data, sedikit konklusi (monitoring)

Fakta: A, B
R1: B ⇒ E   ⇒   A, B, E
R2: A&E ⇒ G   ⇒   A, B, E, G
R3: G ⇒ D (Goal!)

Backward Chaining (Goal-Driven)

Arah: Goal → Fakta (mundur, konvergen)

Cocok: Banyak konklusi, sedikit data (diagnosis)

Goal: D ⇒ butuh G ⇒ butuh A & E
A = fakta (ADA), E = ?
E ⇒ butuh B ⇒ B = fakta (ADA)
Semua sub-goal terbukti ⇒ D BENAR!

Certainty Factor (CF)

Skala: 0.0 (pasti salah) – 1.0 (pasti benar)

CF_hasil = CF_fakta × CF_aturan
// Fakta A (CF=0.90) × Rule (CF=0.75) = 0.675

Explanation Facility

Why? — "Mengapa tanya ini?"
Sistem baca mundur node aktif untuk jelaskan tujuan.

How? — "Bagaimana sampai ke kesimpulan ini?"
Sistem lacak rantai inferensi yang dipakai.

Graceful Failure: Info kurang ⇒ default conclusion teraman (tidak crash).

// Lalu? Tak ada aturan lanjutan

UC Bagian 8: Certainty Factor Deep Dive

Formula Dasar CF

CF(H,E) = MB(H,E) − MD(H,E)
NotasiRentangMakna
MB0 – 1Measure of Belief
MD0 – 1Measure of Disbelief
CF-1.0 – 1.0Keyakinan akhir

CF Gabungan

Single evidence:

CF(H,E) = CF(E) x CF(Rule)

AND (Konjungsi):

CF = min[CF(E1),...,CF(En)] x CF(Rule)

OR (Disjungsi):

CF = max[CF(E1),...,CF(En)] x CF(Rule)

Resolusi Konflik (2 Rule, 1 H)

Keduanya ≥ 0:

CF = CF1 + CF2(1-CF1)

Berbeda tanda:

CF = (CF1+CF2) / (1-min(|CF1|,|CF2|))

Keduanya < 0:

CF = CF1 + CF2(1+CF1)

MB/MD Detail

MB(H,E) = (max[P(H|E),P(H)]-P(H)) / (1-P(H))
MD(H,E) = (P(H)-min[P(H|E),P(H)]) / P(H)
KondisiMBMD
P(H\|E) > P(H)Dihitung0
P(H\|E) < P(H)0Dihitung
P(H\|E) = P(H)00

P(H)=1 ⇒ MB=1; P(H)=0 ⇒ MD=1

Linguistic → Numerik

LinguistikCF
Pasti Tidak-1,0
Kemungkinan Besar Tidak-0,6
Tidak Tahu0,0
Kemungkinan Besar0,6
Hampir Pasti0,8
Pasti1,0

Studi Kasus: FORECAST

Input: Rain=1.0, RainfallLow=0.8, Cold=0.9

R1 (hujan): 1.0 x 0.5 = 0.50
R3 (kering): min(1.0,0.8)x0.6 = 0.48
R4 (kering): min(1.0,0.8,0.9)x0.7 = 0.56

Resolusi Kering:

CF_kering = 0.48 + 0.56(1-0.48) = 0.77
Kering (0.77) > Hujan (0.50)

p(Hi|E) = p(E|Hi) x p(Hi) / Σ p(E|Hk) x p(Hk)

GSP Algorithm: 3-Step Loop

  1. Inisialisasi: Stack = [Goal State sebagai compound goal].
  2. Pecah compound goal → sub-goal individual di atas stack.
  3. Loop sampai stack kosong:
    • Top == TRUE di Current-state → POP
    • Top == sub-goal FALSE → cari operator dgn Add list sesuai, push operator + prekondisinya
    • Top == operator & semua pre TRUE → POP, simpan ke Queue, update Current-state
    • Current-state == Goal → STOP, sukses

McCulloch-Pitts (MCP/TLU)

  • Input/output biner (0/1)
  • w dan θ ditentukan analitis oleh manusia
  • Formula: y = sign(x1w1 + x2w2 - θ)
  • AND: w1=1, w2=1, θ=1,5
  • OR: w1=1, w2=1, θ=0,5
  • GAGAL di XOR (non-linearly separable)

Bukti Kontradiksi XOR (Minsky & Papert 1969)

  1. Dari data (0,0)→0: θ > 0
  2. Dari data (0,1)→1: w2 ≥ θ
  3. Dari data (1,0)→1: w1 ≥ θ
  4. Dari data (1,1)→0: w1 + w2 < θ
  5. Dari (2)+(3): w1+w2 ≥ 2θ > θ (karena θ > 0)
  6. Kontradiksi dengan (4): w1+w2 tidak mungkin < θ dan ≥ 2θ sekaligus.

Solusi: Multilayer network (Backpropagation 1986) — gunakan 2 garis lurus, bukan 1.

HebbNet (Aturan Hebb, 1949)

"Neuron yang aktif bersamaan harus memperkuat sinapsis."

Inisialisasi: wi = 0
Update: wi(new) = wi(old) + xi·t
  • Representasi bipolar (-1/+1) lebih baik dari biner (0/1)
  • AND biner: GAGAL (t=0 → Δw=0)
  • AND bipolar: SUKSES 100% (w1=2, w2=2, w0=-2)

Perceptron (Rosenblatt, 1958)

HebbNet + mekanisme error-based update:

jika y = t:
  wi(new) = wi(old)
jika y ≠ t:
  wi(new) = wi(old) + α·(t-y)·xi
  • α = learning rate (0<α≤1)
  • Epoch = 1 pass lengkap semua data latih
  • Konvergensi = Δw = 0 selama 1 epoch penuh
  • Tidak konvergen untuk non-linearly separable (XOR)

Quick Reference: Simulasi AND Epoch 1 (Perceptron, α=0,2, init w=0)
