5 Inference Rules (WAJIB Hafal)

1. Modus Ponens: α⇒β, α ⊢ β
2. And Elim: α∧β ⊢ α
3. And Intro: α, β ⊢ α∧β
4. Unit Res: α∨β, ¬β ⊢ α
5. Resolusi: α∨β, ¬β∨γ ⊢ α∨γ

Ingat: Implikasi hanya salah kalau anteseden benar, konsekuen salah.

Horn Form (Forward/Backward Chaining)

Definite clause: disjunction dengan tepat satu literal positif.

Horn clause: maksimal satu literal positif.

Horn: ¬A ∨ ¬B ∨ C ≡ A ∧ B ⇒ C
// Semua aturan Wumpus bisa ditulis Horn!

CNF Resolution

Steps to CNF:

  1. Eliminate ⇔ (replace with ⇒ both ways)
  2. Eliminate ⇒ (¬α ∨ β)
  3. Push ¬ inward (De Morgan's)
  4. Distribute ∨ over ∧

Resolution rule: α∨β, ¬β∨γ ⊢ α∨γ

Quantifier Rules of Thumb

∀ pakai ⇒ (implikasi):

∀x (Manusia(x) ⇒ Bernafas(x))

∃ pakai ∧ (konjungsi):

∃x (Manusia(x) ∧ Pintar(x))

Unification + GMP

Unifikasi: mencari substitusi θ yang membuat dua ekspresi identik.

Unify(Knows(John,x), Knows(y,Bob))
θ = { x/Bob, y/John }

Generalized Modus Ponens (GMP):

∀x (α(x) ⇒ β(x)), α(θ)
⊢ β(θ)

7 Langkah Wumpus World (UAS)

  1. Setup Grid & Symbols — Gambar grid 4x4, tentukan simbol proposisi/predikat
  2. Tulis Percept — Tiap langkah: posisi → percept → tetangga
  3. Tulis Aturan Dunia — B ⇔ Pit di tetangga, S ⇔ W di tetangga
  4. Modus Ponens — B ⇒ Pit (disjunction), ¬B ⇒ ¬P semua tetangga (konjungsi)
  5. Eliminasi Resolution — Coret kotak aman dari disjunction, sisa = lokasi mungkin
  6. Cari Bukti Tambahan — Eliminasi pilihan yang tersisa dari percept lain
  7. Tentukan Rute Aman — Jalur terpendek ke emas → kembali → Climb

1. IF-THEN Rules

Struktur: Kondisi → Aksi

THEN [action is go]

SP Bagian 6: Inferensi & Anomali Logika

Forward Chaining (Data-Driven)

Arah: Fakta → Konklusi (maju, divergen)

Cocok: Banyak data, sedikit konklusi (monitoring)

Fakta: A, B
R1: B ⇒ E   ⇒   A, B, E
R2: A&E ⇒ G   ⇒   A, B, E, G
R3: G ⇒ D (Goal!)

Backward Chaining (Goal-Driven)

Arah: Goal → Fakta (mundur, konvergen)

Cocok: Banyak konklusi, sedikit data (diagnosis)

Goal: D ⇒ butuh G ⇒ butuh A & E
A = fakta (ADA), E = ?
E ⇒ butuh B ⇒ B = fakta (ADA)
Semua sub-goal terbukti ⇒ D BENAR!

Certainty Factor (CF)

Skala: 0.0 (pasti salah) – 1.0 (pasti benar)

CF_hasil = CF_fakta × CF_aturan
// Fakta A (CF=0.90) × Rule (CF=0.75) = 0.675

Explanation Facility

Why? — "Mengapa tanya ini?"
Sistem baca mundur node aktif untuk jelaskan tujuan.

How? — "Bagaimana sampai ke kesimpulan ini?"
Sistem lacak rantai inferensi yang dipakai.

Graceful Failure: Info kurang ⇒ default conclusion teraman (tidak crash).

6 Anomali Logika dalam Knowledge Base

1. REDUNDANT Duplikasi aturan

R1: IF starter+suhu THEN mati
R2: IF suhu+starter THEN mati

2. CONFLICTING Konflik keputusan

R1: IF starter+suhu THEN mati
R2: IF starter+suhu THEN berfungsi

3. SUBSUMED Aturan tertelan

R1: starter+suhu+pompa THEN mati
R2: starter+suhu THEN mati
// R1 tak pernah dipakai

4. CIRCULAR Infinite loop

starter → suhu → starter → ...

5. UNNECESSARY IF Premis tak relevan

R1: mesin+starter+klakson THEN setrum
R2: mesin+starter+klakson! THEN setrum
// "klakson" tdk relevan

6. DEAD-END Jalur buntu

IF in1+in2+in3 THEN baterai setrum
// Lalu? Tak ada aturan lanjutan

UC Bagian 8: Certainty Factor Deep Dive

Formula Dasar CF

CF(H,E) = MB(H,E) − MD(H,E)
NotasiRentangMakna
MB0 – 1Measure of Belief
MD0 – 1Measure of Disbelief
CF-1.0 – 1.0Keyakinan akhir

CF Gabungan

Single evidence:

CF(H,E) = CF(E) x CF(Rule)

AND (Konjungsi):

CF = min[CF(E1),...,CF(En)] x CF(Rule)

OR (Disjungsi):

CF = max[CF(E1),...,CF(En)] x CF(Rule)

Resolusi Konflik (2 Rule, 1 H)

Keduanya ≥ 0:

CF = CF1 + CF2(1-CF1)

Berbeda tanda:

CF = (CF1+CF2) / (1-min(|CF1|,|CF2|))

Keduanya < 0:

CF = CF1 + CF2(1+CF1)

MB/MD Detail

MB(H,E) = (max[P(H|E),P(H)]-P(H)) / (1-P(H))
MD(H,E) = (P(H)-min[P(H|E),P(H)]) / P(H)
KondisiMBMD
P(H\|E) > P(H)Dihitung0
P(H\|E) < P(H)0Dihitung
P(H\|E) = P(H)00

P(H)=1 ⇒ MB=1; P(H)=0 ⇒ MD=1

Linguistic → Numerik

LinguistikCF
Pasti Tidak-1,0
Kemungkinan Besar Tidak-0,6
Tidak Tahu0,0
Kemungkinan Besar0,6
Hampir Pasti0,8
Pasti1,0

Studi Kasus: FORECAST

Input: Rain=1.0, RainfallLow=0.8, Cold=0.9

R1 (hujan): 1.0 x 0.5 = 0.50
R3 (kering): min(1.0,0.8)x0.6 = 0.48
R4 (kering): min(1.0,0.8,0.9)x0.7 = 0.56

Resolusi Kering:

CF_kering = 0.48 + 0.56(1-0.48) = 0.77
Kering (0.77) > Hujan (0.50)

UC Bagian 9: Probabilitas Bayes

Teorema Bayes

p(Hi|E) = p(E|Hi) x p(Hi) / Σ p(E|Hk) x p(Hk)
ProbabilitasNotasi
Prior (awal)p(Hi)
Conditionalp(E | Hi)
Posterior (akhir)p(Hi | E)

Syarat: Mutually Exclusive + Conditional Independence

Multi-Evidence

p(Hi|E1..En) =
Πp(Ej|Hi) x p(Hi) / ΣΠp(Ej|Hk) x p(Hk)

Conditional Independence ⇒ kalikan semua p(Ej|Hi)
